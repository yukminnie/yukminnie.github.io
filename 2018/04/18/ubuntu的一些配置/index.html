把本机生成的 id_rsa.pub 添加到服务器 /root/.ssh/authorized_keys 文件中

用 root 用户登录到服务器, 创建 .ssh 目录，创建authorized_keys文件

编辑 authorized_keys 文件, 把刚才生成的 id_rsa.pub 文件里面的内容粘贴进去并保存退出, 这里可以粘贴多个 key, 一行一个
